## Descrição

O BRTrampo é um aplicativo mobile que conecta clientes a prestadores de serviços de forma simples, rápida e confiável.

A plataforma funciona como uma rede de indicações digitais, permitindo que profissionais criem perfis com portfólio, enquanto clientes buscam por serviços com base em localização e categoria.

## Objetivo

Facilitar a contratação de serviços e aumentar a confiança através de avaliações e histórico de profissionais.

## Tecnologias

* Flutter
* Dart
* Firebase
